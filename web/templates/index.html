📄 How the format works

A pack is a small UTF-8 JSON file:

{
  "language": "My Language",   // shown in the picker (required)
  "code": "mylang",            // short unique id (required)
  "direction": "ltr",          // "ltr" or "rtl" (optional)
  "alphabet": ["A", "B", "..."],// keyboard keys; omit for Tamil-like scripts
  "words": [                   // at least one word (required)
    {
      "word": "CAT",           // the word to guess (required)
      "hint": "A pet that says meow", // optional
      "image": "cat"           // optional picture id (see below)
    }
  ]
}
  • Required: language, code, and a non-empty words list (each word needs word).
  • Optional per word: hint and image.
  • alphabet is the on-screen keyboard. Omit it for scripts like Tamil — the keyboard is then built from the word's own characters.
  • image gives a picture reward on a win. See picture ids ▾
